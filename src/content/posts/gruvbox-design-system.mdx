Gruvbox is expressive, but it can become noisy when every accent color competes for attention.

The approach in this portfolio:

- Use warm neutrals for the base surfaces.
- Reserve yellow, aqua, and orange for interaction and emphasis.
- Keep line length and spacing readable for long-form writing.

This keeps the site visually distinctive while still practical for project documentation.
